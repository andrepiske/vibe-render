# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-22

### Added
- Initial release
- Support for rendering webpages as PNG screenshots
- Support for rendering webpages as JPEG screenshots
- Support for rendering webpages as PDF files
- Code API using Ferrum gem
- Configurable viewport, quality, and paper size options

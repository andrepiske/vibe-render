# VibeRender Examples

This directory contains example scripts demonstrating how to use VibeRender.

## Running Examples

Make sure you have the gem installed:

```bash
bundle install
```

Then run any example:

```bash
ruby examples/basic_screenshot.rb
ruby examples/pdf_generation.rb
ruby examples/advanced_options.rb
```

## Examples Included

- `basic_screenshot.rb` - Simple PNG and JPEG screenshot examples
- `pdf_generation.rb` - PDF rendering examples
- `advanced_options.rb` - Advanced usage with custom options
- `local_html.rb` - Rendering local HTML files
